# 📱 Полная документация UI S7 Robotics

Этот документ описывает все страницы, компоненты, формы и пользовательские потоки в приложении S7 Robotics.

---

## 🔐 Аутентификация (Authentication)

### Страница входа (`/` - app/page.tsx)

**Описание**: Главная страница входа/регистрации. Центрированная форма на тёмном фоне с точечным паттерном.

**Состояния формы**:
1. **Вход (Login)**
2. **Регистрация (Register)**  
3. **Восстановление пароля (Forgot Password)**
4. **Подтверждение почты (Email Verification)**

---

#### 1. Форма входа

**Визуальные элементы**:
- Логотип S7 Robotics (80x80px) сверху
- Карточка с пунктирной границей (`border-dashed`)
- Заголовок: "Вход" (белый, 3xl)
- Фон карточки: `#0b0b0b`

**Поля ввода**:
| Поле | Тип | Placeholder | Иконка |
|------|-----|-------------|--------|
| Email | email | "Почта" | ✉️ bi-envelope |
| Пароль | password | "Пароль" | 🔒 bi-lock |

**Кнопки**:
- `[Войти]` - основная кнопка (outline)
- `[Забыли пароль?]` - текстовая ссылка серого цвета
- `[Нет аккаунта? Зарегистрируйтесь]` - переключение на регистрацию

**Нижний колонтитул**:
- Социальные ссылки (SocialPanel)
- "Пользовательские соглашения"
- Версия приложения: 1.0
- Юридическая информация (ИП АМАНТАЙ БАТЫРХАН НАЙМАНҰЛЫ)
- Адреса: Актау, 16 мкр, БЦ ESAL

---

#### 2. Форма регистрации

**Дополнительные поля** (появляются анимированно):

#### 3. Восстановление пароля (Forgot Password)

**3 этапа процесса**:

**Этап 1 - Запрос кода**:
- Поле email (уже заполнено)
- Кнопка `[Отправить код]`

**Этап 2 - Ввод кода**:
- OTP ввод (6 цифр, 2 группы по 3)
```
┌─┐┌─┐┌─┐  ─  ┌─┐┌─┐┌─┐
│ ││ ││ │     │ ││ ││ │
└─┘└─┘└─┘     └─┘└─┘└─┘
```
- Кнопка `[Подтвердить код]`

**Этап 3 - Новый пароль**:
| Поле | Placeholder |
|------|-------------|
| Новый пароль | "Новый пароль" |
| Подтверждение | "Повторите пароль" |

- Минимум 8 символов
- Кнопка `[Сменить пароль]`

#### 4. Подтверждение почты (Email Verification)

**Компонент**: `components/auth/email-verification.tsx`

**Описание**: Модальное окно для ввода 6-значного кода подтверждения.

**Элементы**:
- Заголовок: "Подтверждение почты"
- Текст: "Мы отправили код подтверждения на {email}"
- OTP поля (6 слотов, размер 12x12)
- Кнопка `[Подтвердить]` (disabled пока код ≠ 6 символов)
- Кнопка `[Отправить заново]`
- Кнопка `[Назад]`

**Стили кнопок**:
```css
/* Основная кнопка */
bg-[#0f0f0f] border-[#1a1a1a] 
hover:bg-[#141414] hover:border-[#2a2a2a]
rounded-full

/* Вторичные кнопки */
border-[#1f1f1f] text-[#a7a7a7]
hover:bg-[#141414] hover:border-[#2a2a2a] hover:text-white
```

## 🔔 Уведомления

### Toast (всплывающие)

**Варианты**:
- `default` - обычное (серый фон)
- `success` - успех (зелёный акцент)
- `destructive` - ошибка (красный акцент)

**Пример**:
```tsx
toast({
  title: "Успешно",
  description: "Действие выполнено",
  variant: "default" // или "destructive"
})
```

### Notifications (components/notifications/)

**Типы уведомлений**:
- Новое занятие
- Отмена занятия
- Новое достижение
- Системное сообщение
