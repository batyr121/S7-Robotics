### 1. Роль: УЧЕНИК (Student)
Веб-страница разрабатывается для управления обучением со стороны ученика.
**Вкладки:**
1.  **Главная (Dashboard):**
    * Отображается актуальное расписание.
    * Блок "Мой курс": если ученик записан, показывается информация о текущем курсе. Если не записан — блок пустой или призыв к действию.
2.  **Курсы (Courses):**
    * Список актуальных офлайн-курсов центра.
    * Возможность покупки курса онлайн.
    * Карточка курса должна содержать: Описание, Цена, Программа курса (презентации нет, только текст/список).
3.  **Расписание (Schedule):**
    * Календарь занятий.
    * Примечание: расписание не меняется (статичное или фиксированное).
4.  **Профиль (Profile):**
    * Личные данные (не меняются учеником, только просмотр).
    * **Важно:** Добавить вкладку/раздел "Сертификаты" (список полученных сертификатов).

### 2. Роль: МЕНТОР (Mentor)
Интерфейс для преподавателей.
**Вкладки:**
1.  **Главная (Dashboard):**
    * Уведомления о новых открытых группах.
    * Краткое расписание на сегодня.
    * Информация о текущей зарплате (начисления).
2.  **Группы (Groups):**
    * Список курируемых групп.
    * Информация по каждой группе: количество детей.
    * Статус оплат учеников (кто оплатил абонемент, кто нет).
    * Программа обучения для группы.
3.  **Кошелек (Wallet):**
    * Баланс ментора.
    * История выплат.
    * Детализация начислений.
4.  **Расписание (Schedule):**
    * Полное расписание уроков ментора.
5.  **Профиль (Profile):**
    * Уровень квалификации (грейды).
    * Сертификаты ментора.
    * Ранг (система геймификации для сотрудников).

### 3. Роль: РОДИТЕЛЬ (Parent)
Интерфейс для родителей, управляющих обучением своих детей.
**Вкладки:**
1.  **Главная (Dashboard):**
    * Новости (отправляются админом).
    * Актуальные скидки.
    * Состояние абонемента (активен/не активен).
2.  **Дети (Children):**
    * Функционал добавления детей (привязка по почте).
    * Покупка абонементов для каждого ребенка.
3.  **Расписание (Schedule):**
    * Объединенное расписание уроков всех детей.
    * **Важно:** В расписании должны отображаться даты оплат (напоминания, когда платить).
4.  **Мастер-классы:**
    * Список доступных мастер-классов для записи.
5.  **Bytesize (Обучение для родителей):**
    * Отдельный раздел с короткими видео (Reels/Shorts формат) специально для родителей (про воспитание, IT и т.д.).
6.  **Магазин (Shop/Loyalty):**
    * Бонусная система.
    * Бонусы начисляются за посещение мероприятий и за покупки обучения детей.
7.  **Аналитика:**
    * Прогресс детей (оценки, посещаемость).

### 4. Роль: КУРАТОР / АДМИН (Curator/Admin)
Управленческая роль.
**Функционал:**
* **Управление (Management):** Общий доступ к редактированию расписания, новостей, пользователей.
* **Bytesize для менторов:** Загрузка и просмотр обучающих видео специально для педагогов (повышение квалификации).
* **Запись на курсы:** Возможность записаться на курсы для педагогов (пометка: "Бесплатно!").
